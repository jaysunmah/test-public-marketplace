# Example Skill

This skill demonstrates the skill format for Claude Code plugins.

## When to Use

Activate this skill when:
- The user asks about plugin development patterns
- The user needs an example of how skills work
- The user wants to understand the plugin system

## Instructions

1. Explain the plugin structure in one short overview.
2. Point to one concrete file path the user can copy as a starting template.
3. Provide a minimal implementation checklist (build, validate, and test).
